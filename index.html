Get Started · 시작하기

실습 준비 (3단계)

  1. Anthropic API Key 발급 — console.anthropic.com에서 키를 만들고 소액($5~)을 충전합니다.
  2. Colab에서 열기 — 위 노트북의 “Colab에서 열기” 버튼을 클릭합니다(구글 로그인 필요).
  3. 키 입력 후 실행 — 노트북 맨 위 ‘환경 셋업’ 셀에서 API Key를 입력하고, 셀을 위에서 아래로 차례로 실행하세요. 실습 기본 모델은 claude-sonnet-4-6입니다.
